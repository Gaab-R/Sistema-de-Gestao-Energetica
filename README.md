# SAPEX – Sistema de Gestão Energética

Sistema para monitoramento e análise de consumo de energia em edificações.

A proposta é simular um ambiente inteligente, coletando dados, analisando padrões e sugerindo ações para melhorar a eficiência energética.

---

## Sobre

O sistema permite:

- Monitorar variáveis como temperatura, umidade e consumo  
- Analisar dados de uso de energia  
- Sugerir ações de economia  
- Automatizar dispositivos  

Inspirado em conceitos de IoT aplicados a eficiência energética.

---

## Funcionalidades

- Monitoramento em tempo real  
- Análise de dados com IA  
- Automação de dispositivos  
- Integração com ThingSpeak  
- Armazenamento em banco de dados  
- Dashboard com gráficos  

---

## Tecnologias

- Python (Flask)  
- MySQL  
- Pandas  
- Transformers  
- HTML, CSS e JavaScript  
- Chart.js
